## 파이썬 기초 강의



- Polyglot 프로그래머
  - 활용되는 부분마다 다르게 적용이 되고있다.
  - 해당 도메인에 적합한 언어와 프레임워크를 배워서 행한다.
- 프로그래밍을 배울 때 가장 중요한 것 3가지

1. 왜?

2. 개념?

3. 구조?

- What is Programming Language?
  - 컴퓨터 – 계산과 저장을 잘하는 기계

  - 프로그래밍 – 명령어의 집합, 지시사항의 모음

  - 언어 – 말

    => 컴퓨터에게 뭔가를 시킬 때 쓰는 말

- 인터프리터

- 어셈블리어 -> C언어 -> Python

- 언어? == 약속

- 문법? == 문장을 만드는 법

- **컴퓨터 문법은 3형식 밖에 없다.** 
  - If (조건)
  - While (반복)
  - =(저장)

- 형식? == 조사가 없어서 정해주는 순서이다.
