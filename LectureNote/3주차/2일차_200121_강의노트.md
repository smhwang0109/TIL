- 저장
  - 어떻게 (연산자) : =
  - 무엇을(데이터타입 == 자료형)
    - 숫자
    - 글자
    - bool
  - 어디에 (변수 / 컨테이너)
    - 변수
    - 시퀀스형 자료(순서가 있는) 
      - immutable : string, tuple, range
      - mutable : list
    - 비시퀀스형 자료(순서가 없는)
      - immutable : dict, set



- 식(expression) & 문(statement)

  - 식(expression)

    = value(값) & operator(연산자)

    = evaluate(연산자를 통한 값의 평가)

    -> **변수에 할당 가능 (== 변수에 바인딩 가능)**

  

  - 문(statement)

    = expression으로 구성

    = 세미콜론 포함 ( : )

    -> **변수에 할당 불가**
