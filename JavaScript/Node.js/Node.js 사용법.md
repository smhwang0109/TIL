# Node.js 설치

```bash
node --version
V12.16.3
```



npm : node에서의 pip 같은 것

```bash
npm install -g @vue/cli
```

-g는 global -> 하지만 공식 문서에 -g라고 되어 있는 것들만 이렇게 설치하는 것이다.



vue 하고 이제 쓸 수 있다. 

vue 프로젝트 만들기

```bash
vue create 만들폴더명

cd 만든 폴더명

-- 라우터 추가
vue add router

npm run serve

-- vuex 추가
vue add vuex
```



package 설치

```bash
npm i 설치할패키지명
```



개발이 끝난 후

```bash
npm run build
```
