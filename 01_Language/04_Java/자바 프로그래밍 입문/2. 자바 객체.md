# 2. 자바 객체

## 1_객체 지향 프로그래밍이란

### 객체란?

세상에 존재하는 모든 것을 뜻하며, 프로그래밍에서 속성과 기능을 가지는 프로그램 단위이다.



### 클래스란?

객체를 생성하기 위한 틀로 모든 객체는 클래스로부터 생성된다. (붕어빵 틀로 생각)



### 클래스 구성요소

클래스는 속성(멤버 변수)과 기능(메소드)로 구성된다.



## 2_클래스 제작과 객체 생성

```java
// Grandeur.java

package testPjt;

public class Grandeur {
	
	// 멤버 변수(속성)
	public String color;
	public String gear;
	public int price;
	
	// 생성자
	public Grandeur() {
		System.out.println("생성자 입니다.");
	}
    
    // 생성자 여러개도 가능
    public Grandeur(String c, String g, int p) {
        System.out.println("2번째 생성자 입니다.");
        color = c;
        gear = g;
        price = p;
    }
	
	// 메서드(기능)
	public void run() {
		System.out.println("run 메서드(기능) 입니다.");
	}
	
    // void 는 return 값이 없는 경우
	public void stop() {
		System.out.println("stop 메서드(기능) 입니다.");
	}
	
	public void info() {
		System.out.println("Grandeur color : " + color);
		System.out.println("Grandeur gear : " + gear);
		System.out.println("Grandeur price : " + price);
	}
	
}

// MainClass.java

package testPjt;

import java.util.Scanner;

public class MainClass {
	public static void main(String[] args) {
		
		Grandeur myCar1 = new Grandeur();
		myCar1.color = "red";
		myCar1.gear = "auto";
		myCar1.price = 100;
		
		myCar1.run();
		myCar1.stop();
		myCar1.info();
        
        Grandeur myCar2 = new Grandeur("yellow", "manual", 50);
		
		myCar2.run();
		myCar2.stop();
		myCar2.info();
		
	}
	
	
}

```

### 매개변수

```java
// ChildClass.java

package testPjt;

public class ChildClass {

	public String name;
	public String gender;
	public int age;
	
	public ChildClass() {
		System.out.println("ChildClass constructor");
	}
	
	public void setInfo(String n, String g, int a) {
		name = n;
		gender = g;
		age = a;
	}
	
	public void getInfo() {
		System.out.println("name : " + name);
		System.out.println("gender : " + gender);
		System.out.println("age : " + age);
	}
	
}

// MainClass.java

package testPjt;

import java.util.Scanner;

public class MainClass {
	public static void main(String[] args) {
		
		ChildClass child1 = new ChildClass();
		child1.setInfo("Soom", "M", 27);
		child1.getInfo();
	}
	
	
}
```



### 중복 메서드(overloading)

> 중복해서 메서드를 선언해도 되지만 매개변수를 다르게 해줘야 한다.

```java
public void getInfo() {
    System.out.println("-- getInfo - I --")
}

public void getInfo(int x, int y) {
    System.out.println("-- getInfo - II --")
}

public void getInfo(String s1, String s2) {
    System.out.println("-- getInfo - III --")
}
```

## 4_객체와 메모리

### 메모리에서 객체 생성(동적 생성)

- 객체는 메모리에서 동적으로 생성되며,
- 객체가 더 이상 필요 없게 되면 GC(Gabage Collector)에 의해서 제거된다.



### 레퍼런스

- 생성한 객체의 주소를 변수에 저장하는 것을 레퍼런스라고 한다.

```java
// MainClass.java

package testPjt;

import java.util.Scanner;

public class MainClass {
	public static void main(String[] args) {
		
		ObjectClass obj1 = new ObjectClass();
		ObjectClass obj2 = new ObjectClass();
		
		if(obj1 == obj2) {
			System.out.println("같다.");
		} else {
			System.out.println("다르다.");
		}
		
		System.out.println(obj1);
		System.out.println(obj2);
	}
	
	
}
```



### null과 NullPoint Exception

```java
// MainClass.java

package testPjt;

import java.util.Scanner;

public class MainClass {
	public static void main(String[] args) {
		
		ObjectClass obj1 = new ObjectClass();
		
		obj1 = null;
		
		System.out.println(obj1);
		
		// NullPointerException
		// obj1.getInfo();
		
		obj1 = new ObjectClass();
		
		// 다시 새로운 객체를 가리킨다.
		obj1.getInfo();
		
	}
	
	
}
```
